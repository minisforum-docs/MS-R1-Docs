# MS-R1

## 소개

MS-R1은 많은 안드로이드 스마트폰과 동일한 아키텍처를 공유하는 ARM 기반 미니 PC입니다.
이러한 아키텍처 공통성 덕분에 MS-R1은 기존 x86 안드로이드 에뮬레이터보다 훨씬 높은 호환성으로 Docker 환경에서 안드로이드를 실행할 수 있습니다.

MS-R1은 ARM Immortalis-G720 MC10 GPU가 통합된 ARM V9 아키텍처 기반 CIX CP8180 프로세서를 탑재하고 있습니다.
CPU의 멀티코어 성능은 인텔 Core i5-12450H와 유사하며, GPU 성능은 AMD Radeon 680M 내장 그래픽과 유사한 수준입니다.

CP8180는 AV1,H265,H264 영상 디코딩과 H265,H265 영상 인코딩을 지원합니다.

CP818는 KVM을 지원하여 매우 효율적인 가상화를 구현할 수 있습니다.
PCIe 패스스루 기능 역시 지원합니다.

### I/O 포트

#### 전면 I/O

| 포트 / 버튼      | 비고                                                     |
| ---------------- | -------------------------------------------------------- |
| 1× 전원 버튼     | MS-R1을 켜고 끕니다. 5초간 눌러 강제종료 할 수 있습니다. |
| 1× 3.5mm 콤보 잭 | 표준 4핀 콤보 잭입니다 (CTIA 표준).                      |
| 1× USB 3.2 Gen 2 | 10 Gbps Type-A USB3 포트                                 |
| 2× USB 2.0       | 480 Mbps USB2 Type-A 포트                                |

#### 후면 I/O

| 포트 / 버튼      | 비고                                                                                      |
| ---------------- | ----------------------------------------------------------------------------------------- |
| 2× USB 2.0       | 480 Mbps USB2 Type-A 포트                                                                 |
| 2× RJ45          | 10 Gbps RJ45 이더넷 포트 (RTL8127 기반)                                                   |
| 2× Type-C        | 완전 기능형 Type-C 포트 (USB 데이터 전송 최대 10 Gbps, DP 1.4). USB4는 지원하지 않습니다. |
| 1× HDMI          | HDMI 2.0 포트                                                                             |
| 2× USB 3.2 Gen 2 | 10 Gbps Type-A USB3 포트                                                                  |
| 1× 전원 포트     | 5.5×2.5 mm 19V 전원 입력                                                                  |

#### 내장 I/O

##### 후면

| 포트 / 버튼               | 비고                                                                                  |
| ------------------------- | ------------------------------------------------------------------------------------- |
| 1× 40핀 GPIO              | 핀 정의 미정                                                                          |
| 1× 자동 전원 스위치       | ON: 전원이 연결되면 자동으로 켜집니다.                                                |
| BIOS 플래시 핀 & UART1 핀 | 플래시 메모리 핀이 BIOS 플래시에 연결됩니다. UART1 핀은 메인보드에 표시되어 있습니다. |
| 1× U.2 전원               | U.2용 12V 전원                                                                        |
| 1× M.2 E-key              | PCIe 4.0 x2, USB 지원                                                                 |
| 1× M.2 M-key              | PCIe 4.0 x4                                                                           |
| 1× TPM 핀                 | TPM 전용 커넥터                                                                       |
| 1× 5V 팬 커넥터           | 5V, SSD용 팬                                                                          |
| 1× RTC 커넥터             | 실시간 시계 커넥터                                                                    |

##### CPU 측

| 포트 / 버튼  | 비고                                     |
| ------------ | ---------------------------------------- |
| 1× UART2     | 메인보드에 UART2 핀이 표시되어 있습니다. |
| 1× CPU 팬    | CPU 쿨링 팬 커넥터                       |
| 1× PCIe 포트 | PCIe 4.0 x8 (레인 분할 미지원)           |
